REST API 端点
POST /api/auth/login/ 管理员登录 → JWT Token
GET /api/albums/ 相册列表
POST /api/albums/ 创建相册 🔐
GET /api/albums/{id}/ 相册详情（含照片列表）
DELETE /api/albums/{id}/ 删除相册 🔐
POST /api/upload/presigned/ 获取 OSS 上传凭证 🔐
POST /api/photos/ 保存照片元数据 🔐
DELETE /api/photos/{id}/ 删除照片 🔐
GET /api/photos/?ordering=-taken_at 时间线视图（分页）
🔐 需要 JWT 认证的端点

关键设计要点

  • 上传走 OSS 直传 — 前端获取 Presigned URL 后直接上传到 OSS，不经 Django，不占服务器带宽
  • 缩略图用 OSS 图片处理 — 通过 URL 参数 ?x-oss-process=image/resize,w_400 实时获取缩略图，无需后端处理
  • 照片以 taken_at 排序 — 上传时读取 EXIF 拍摄时间，无 EXIF 则用上传时间
  • 管理端点统一加 JWT — 查询类接口公开，写操作需认证
