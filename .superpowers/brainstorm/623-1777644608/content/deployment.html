部署清单

  • 阿里云 ECS 或轻量应用服务器
  • 域名 + SSL 证书（Let's Encrypt 免费）
  • 阿里云 OSS Bucket + AccessKey
  • 高德地图 Web 服务 API Key
  • Nginx + Gunicorn + PostgreSQL

安全注意事项

  • OSS Bucket 设为私有，仅通过 Presigned URL 访问
  • API Key 通过环境变量注入，不提交仓库
  • Django SECRET_KEY 独立随机生成
  • 管理端 JWT 有效期设置合理（如 7 天）
  • Nginx 配置 rate limiting 防暴力破解
